About

The Portfolio is a simple one page Bootstrap theme for traditional artists or freelancers. A responsive grid style with hover effects and modal windows showcasing your artwork full size.

This simple template is the perfect starting point. The files include HTML, CSS, and JavaScript files. The Portfolio is free to use for personal projects.
